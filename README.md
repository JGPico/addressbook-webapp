# addressbook-webapp
A simple contact list written with vanilla JS

## Features
- Add new contacts with name, email, phone, and address
- View all contacts in a clean, card-based layout
- Search contacts by any field
- Delete contacts
- Data persists in browser localStorage

## Project Structure
```
addressbook-webapp/
├── index.html      # Main HTML structure
├── styles.css      # Styling and layout
├── app.js          # Application logic and functionality
└── README.md       # Project documentation
```

## Getting Started
1. Open `index.html` in a web browser
2. Start adding contacts using the form
3. Use the search bar to filter contacts

## Technologies
- Vanilla JavaScript (ES6+)
- HTML5
- CSS3 (with modern features like Grid and Flexbox)
- LocalStorage API for data persistence
